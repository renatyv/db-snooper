How many times has each robot failed or timeouted to MOVE at '2026-06-08'. Get robot id, level, absolute position in mms. Order by number of errors in descending order.

# refence query
```sql
SELECT
    ah.robot_id, rsh.level, rsh.x_mm, rsh.y_mm, COUNT(*) AS error_count
FROM
    action_history ah
    JOIN action_status_history ash ON ash.action_history_id = ah.id
    JOIN robot_state_history rsh ON rsh.id = ash.state_history_id
WHERE
    ah.command = 'MOVE'
    AND ash.action_status IN ('FAILED', 'FAILED_TIMEOUT')
    AND ah.start_time >= '2026-06-08'
    AND ah.start_time < '2026-06-09'
GROUP BY
    ah.robot_id, rsh.level, rsh.x_mm, rsh.y_mm
ORDER BY error_count DESC;
```
