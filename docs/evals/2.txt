# Task
 Daily per-robot/per-command reliability aggregates for June 8 2026 for commands `MOVE`, `PUT_BOX`, `ROTATE_WHEELS`, and `TAKE_BOX`, counting distinct `DONE`, `FAILED`, and `FAILED_TIMEOUT` actions only when there are at least 15 commands in that bucket.

return
robot id, command, number of successful actions, number of failed, number of timeouted

# expected result
```
5	MOVE	451	0	0
5	PUT_BOX	118	0	0
5	ROTATE_WHEELS	256	0	0
5	TAKE_BOX	117	0	2
7	MOVE	442	17	0
7	PUT_BOX	124	0	1
7	ROTATE_WHEELS	264	0	0
7	TAKE_BOX	121	2	6
```

# refence query
```sql
SELECT
    `ah`.`robot_id`,
    `ah`.`command`,
    COUNT(DISTINCT IF(
        `ash`.`action_status` = 'DONE',
        `ash`.`action_history_id`,
        NULL
    )) AS `done`,
    COUNT(DISTINCT IF(
        `ash`.`action_status` = 'FAILED',
        `ash`.`action_history_id`,
        NULL
    )) AS `failed`,
    COUNT(DISTINCT IF(
        `ash`.`action_status` = 'FAILED_TIMEOUT',
        `ash`.`action_history_id`,
        NULL
    )) AS `failed_timeout`
FROM `action_status_history` AS `ash`
JOIN `action_history` AS `ah`
    ON `ah`.`id` = `ash`.`action_history_id`
JOIN `robot` AS `r`
    ON `r`.`id` = `ah`.`robot_id`
    AND ah.start_time >= '2026-06-08'
    AND ah.start_time < '2026-06-09'
WHERE `ash`.`time` >= '2026-06-08'
  AND `ash`.`time` < '2026-06-09'
  AND `ash`.`action_status` IN ('DONE', 'FAILED', 'FAILED_TIMEOUT')
  AND `ah`.`command` IN ('PUT_BOX', 'TAKE_BOX', 'MOVE', 'ROTATE_WHEELS')
GROUP BY
    DATE_FORMAT(`ash`.`time`, '%Y-%m-%d 00:00:00'),
    `ah`.`robot_id`,
    `r`.`level`,
    `ah`.`command`
HAVING COUNT(DISTINCT `ash`.`action_history_id`) >= 15
ORDER BY
    `r`.`level`,
    `ah`.`robot_id`,
    DATE_FORMAT(`ash`.`time`, '%Y-%m-%d 00:00:00'),
    `ah`.`command`;
```
